Bloco 1

O que é uma
tool?

Tool é uma capacidade extra que você libera para o agente, e que ele decide usar quando faz sentido.

Explique que ferramenta ligada não significa ferramenta usada. O agente avalia se precisa.

Analogia

O estagiário inteligente

Sem ferramentas

Trancado numa sala sem internet, sem celular e sem calculadora.

Responde de cabeça e às vezes chuta.

Com ferramentas

Com Google, calculadora, arquivos e editor de imagem.

Pesquisa, calcula, lê e entrega arte.

Essa analogia ajuda muito. Não use termos técnicos ainda.

Quem decide?

O agente escolhe
quando usar

2 + 2?

Provavelmente não precisa pesquisar nem rodar código.

Cotação do dólar agora?

Precisa de busca ou fonte atual.

Reforce a sacada: mais tools não é automaticamente melhor. Ligue só o que faz sentido para o projeto.

Bloco 2

🔎 Web Search
Grounding

Quando ligado, a IA pesquisa no Google antes de responder, lê resultados e usa fontes atuais.

Diga o nome do toggle: “Grounding with Google Search”.

Quando ligar

Web Search serve para hoje

  • Fatos atuais.
  • Preços e cotações.
  • Notícias e mercado.
  • Docs atualizadas.
  • Bibliotecas recentes.
  • Verificar pacote real.

Lembre o alerta da Aula 1: a IA inventa nome de pacote. Web Search ajuda a verificar se existe.

Quando NÃO ligar

Busca também distrai

Evite quando

  • Você quer só código limpo.
  • O assunto é genérico.
  • Você não precisa de fonte atual.

Por quê?

A resposta pode ficar mais lenta, verbosa e cheia de citações desnecessárias.

Esse slide evita a ideia de “deixar tudo ligado sempre”.

Bloco 4

🐍 Code Execution
IA que calcula

Quando ligado, a IA pode escrever e rodar Python em um sandbox para checar resultados antes de responder.

Explique que LLM é ruim de conta, mas ótima em escrever código para calcular.

Problema enorme

LLM erra conta

Sem execução

Ela tenta calcular de cabeça e pode errar a casa decimal.

Com execução

print(0.18 * 7842) # 1411.56

Faça a demonstração com 18% de 7.842. É simples e didático.

Quando ligar

Code Execution serve para dados

  • Conta e porcentagem.
  • Média, mediana, top N.
  • Gráficos.
  • Validar JSON.
  • Testar regex.
  • Combinar CSVs.

Mostre que não é só matemática escolar. É análise de dados e validação prática.

Hackathon

Casos de uso com Python

Pega essa planilha de votos e devolve um gráfico de pizza dos 3 mais votados. Gera 1.000 nomes fictícios brasileiros para popular meu app de demo. Calcula o ROI dessa simulação para cada cenário e monta uma tabela.

Se tiver tempo, use uma lista pequena e peça média ou gráfico. É uma demo segura.

Limitação honesta

Sandbox não tem
internet

Ótimo para

Dados que você forneceu: CSV, números, JSON, texto, arquivos.

Ruim para

“Baixe esse CSV de uma URL e analise”. Para isso, combine com busca ou forneça o arquivo.

Limitação honesta aumenta confiança. Ferramenta certa para problema certo.

MCP

USB-C da IA

O que é?

Um padrão para conectar agentes a serviços externos usando o mesmo encaixe.

DIAGRAMA
Agente → MCP → Gmail, GitHub, Postgres, Slack, Navegador

Analogia: antes cada integração era um cabo diferente; MCP tenta padronizar o encaixe.

O que dá para fazer

Agente mexendo
em serviços

  • Ler e responder Gmail.
  • Abrir PRs no GitHub.
  • Consultar Postgres.
  • Controlar navegador.
  • Conversar com Slack.
  • Atualizar Notion/Linear.

Reforce: para hackathon, provavelmente overkill. Mas se o projeto é “agente que faz coisas em app X”, MCP é caminho.

Skills

Habilidades empacotadas

O que é?

Uma pasta com instruções, scripts, exemplos e assets para ensinar uma habilidade específica ao agente.

skill/ ├── SKILL.md ├── scripts/ ├── examples/ └── assets/

Explique que o agente descobre a skill quando ela é relevante e segue as instruções dela.

Exemplos de Skills

Relatórios e arquivos

PDF

Criar, mesclar, extrair e manipular PDFs.

XLSX

Ler e gerar planilhas Excel complexas.

DOCX

Gerar documentos Word formatados.

PPTX

Criar apresentações estruturadas.

Conecte com hackathon: se o projeto gera relatório em PDF/Excel/Word, skills podem acelerar muito.

Matriz de decisão

Preciso...Tool
Pesquisar fatos atuais🔎 Grounding
Ler PDF, imagem, vídeo ou CSV📄 Anexar arquivo
Fazer cálculo, gráfico ou dados🐍 Code Execution

Esse é o slide de maior utilidade prática. Pode virar colinha do hackathon.

Checklist

Antes de prompar

  • Envolve mundo real atual? Ligue Grounding.
  • Tenho arquivo útil? Anexe.
  • Tem conta ou dados? Ligue Code Execution.
  • Preciso de visual? Use modelo de imagem.
  • Não preciso de nada disso? Desligue tudo.
  • Tool certa, prompt certo, spec certa.

Reforce: desligar tools quando não precisa deixa o fluxo mais rápido e direto.
